# Шаблон Личного Файла Специалиста

Статус: активный шаблон.

Цель: чтобы каждый специалист жил как отдельный проектный файл и вызывался по инструкции, а не “из воздуха”.

---

---

## Как Это Использовать

Логика такая:

1. создаём один файл на специалиста;
2. заполняем его дефолтной ролью;
3. привязываем к этапам;
4. потом докладываем чемоданчик;
5. потом уже тестируем на картошке или на живом проекте.

---

## Главное Правило

Специалист не должен существовать только как название.

У него должен быть:

- личный файл;
- понятная зона;
- понятный тип участия;
- понятный формат результата;
- понятный чемоданчик.
