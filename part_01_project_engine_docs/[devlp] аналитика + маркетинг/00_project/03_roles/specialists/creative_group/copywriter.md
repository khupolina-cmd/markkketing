# Копирайтер

## 1. Кто это

- группа: креативная
- главная функция: написать текст слайда внутри уже заданной воронки
- где нужен: этап 7A

## 2. За что отвечает

- заголовки;
- подзаголовки;
- микротекст;
- доказательные формулировки;
- язык покупателя без кликбейта.

## 3. Куда не лезет

- не меняет порядок слайдов;
- не меняет роль слайда;
- не выбирает стратегию;
- не усиливает обещание без proof.

## 4. Тип участия по этапам

| Этап | Тип участия | Что делает | Что отдаёт |
|---|---|---|---|
| 7A | owner | пишет copy внутри locked funnel | copy layer по каждому слайду |

## 5. Главные вопросы специалиста

- это понятно за 2-3 секунды?
- это выполняет обещание заголовка?
- это звучит человечески?
- claim привязан к proof?

## 6. Главные запреты

- не использовать фальшивую срочность;
- не писать "лучший", "идеальный", "для всех" без доказательства;
- не превращать текст в подпись к картинке;
- не спорить с матрицей и отзывами.
