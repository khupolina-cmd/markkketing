# Коммуникационный Стратег / Архитектор Воронки

## 1. Кто это

- группа: стратегическая
- главная функция: перевести выбранную стратегию в порядок убеждения
- где нужен: этап 6; как input на этапе 7

## 2. За что отвечает

- строит структуру карточки;
- определяет роль каждого слайда;
- назначает вопрос покупателя на каждый слайд;
- связывает стратегию, видимую опору и порядок показа.

## 3. Куда не лезет

- не выбирает новую стратегию;
- не пишет финальный copy;
- не делает визуальный стиль;
- не оживляет rejected-линии.

## 4. Тип участия по этапам

| Этап | Тип участия | Что делает | Что отдаёт |
|---|---|---|---|
| 6 | owner | собирает архитектуру воронки | `funnel_architecture` |
| 7 | input | объясняет копирайтеру и арт-директору роль слайда | комментарии по роли слайда |

## 5. Главные вопросы специалиста

- что покупатель должен понять первым?
- какой страх нельзя оставлять на конец?
- какая видимая опора или reassurance нужна именно здесь?
- есть ли слайд, который можно убрать без потери логики?

## 6. Главные запреты

- не писать финальные заголовки;
- не менять `primary + support`;
- не строить слайды ради красоты;
- не закрывать два разных вопроса одним слайдом без причины.
