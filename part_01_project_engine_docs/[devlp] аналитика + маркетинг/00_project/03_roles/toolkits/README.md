# Чемоданчики

Здесь живут чемоданчики специалистов.

Перед любым пополнением сначала сверяемся с каноническим словарем:

- [`CANONICAL_TERMINOLOGY_AND_ELEMENTS_RU.md`](../CANONICAL_TERMINOLOGY_AND_ELEMENTS_RU.md)
- [`TOOLKIT_ROUTING_AND_PRIORITY_PROTOCOL_RU.md`](../TOOLKIT_ROUTING_AND_PRIORITY_PROTOCOL_RU.md)

Для каждого специалиста держим минимум два файла:

- `default_tools.md`
- `user_modifiers.md`

Первый слой дает базовую профессиональную практику.

Второй слой дает твои личные правила и override-модификаторы.

Короткое обязательное правило:

```text
сначала нормализуем язык и элементы,
потом докладываем новые правила в чемоданчик
```
