# Дефолтные инструменты: Арт-директор

## 1. Иерархия внимания

Когда использовать: на каждом слайде.

Вопрос:

```text
что человек должен увидеть первым, вторым и третьим?
```

Запрещает: визуальную кашу.

---

## 2. Визуальное доказательство

Когда использовать: если слайд обещает свойство продукта.

Вопрос:

```text
можно ли это показать, а не просто написать?
```

Запрещает: декоративный визуал вместо доказательства.

---

## 3. Визуальная честность

Когда использовать: при любом красивом усилении.

Вопрос:

```text
не обещает ли картинка больше, чем продукт выдерживает?
```

Запрещает: ложное ожидание.

---

## 4. Ограничение стилевой матрицей

Когда использовать: после выбора primary + support.

Вопрос:

```text
визуал соответствует выбранной линии из авторской матрицы?
```

Запрещает: уходить в чужой стиль.
