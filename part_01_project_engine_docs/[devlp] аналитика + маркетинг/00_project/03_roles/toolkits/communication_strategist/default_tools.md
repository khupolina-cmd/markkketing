# Дефолтные инструменты: Коммуникационный стратег

## 1. Лестница вопросов покупателя

Когда использовать: при сборке порядка слайдов.

Вопрос:

```text
какой вопрос покупателя должен быть закрыт до следующего слайда?
```

Запрещает: ставить слайды в порядке "как удобно рассказать продавцу".

---

## 2. Связка возражения и доказательства

Когда использовать: если в матрице есть опасные минусы.

Вопрос:

```text
какой proof нужен, чтобы это сомнение не осталось висеть?
```

Запрещает: закрывать серьёзный страх красивой формулировкой.

---

## 3. Один слайд - одна работа

Когда использовать: на каждом слайде.

Вопрос:

```text
какую одну работу делает этот слайд?
```

Запрещает: один слайд = и состав, и подарок, и эмоция, и акция.

---

## 4. Тест необходимости слайда

Когда использовать: перед передачей в этап 7.

Вопрос:

```text
если убрать этот слайд, логика покупки ломается?
```

Запрещает: добавлять слайд "для полноты".
