# Дефолтные инструменты: Концепт-разработчик

## 1. Механика служит воронке

Когда использовать: при любой идее механики.

Вопрос:

```text
эта механика помогает пройти воронку или просто украшает?
```

Запрещает: спецход ради спецхода.

---

## 2. Сценарная драматизация

Когда использовать: если нужно показать сценарий использования.

Вопрос:

```text
какую сценку можно показать, чтобы покупатель быстрее понял смысл?
```

Запрещает: абстрактную красивость без сценария.

---

## 3. Доказательство, а не театр

Когда использовать: если механика становится слишком эффектной.

Вопрос:

```text
это всё ещё proof или уже театр вместо proof?
```

Запрещает: подмену доказательства драматизацией.
