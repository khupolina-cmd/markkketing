# Product Passport Input

Статус: сырой ввод для ИИ.

Это можно заполнять неаккуратно.

Сюда можно вставить всё, что быстро копируется, и добавить голосовые заметки по обещаниям с картинок.

---

## 1. Минимум, Который Нужен

```text
product_key:
product_role: hero / direct / indirect
marketplace:
product_url:
snapshot_date:
```

---

## 2. Быстрые Цифры

```text
current_price:
old_price_if_visible:
volume_weight_count:
rating:
reviews_count:
purchases_count_if_visible:
```

---

## 3. Всё, Что Скопировалось Со Страницы

Вставить как есть:

```text
copied_page_text:
```

---

## 4. Обещания С Картинок / Слайдов

Можно продиктовать голосом коротко.

Не нужно красиво.

```text
image_claims_voice_notes:
- 
- 
- 
```

Примеры:

```text
- на первой картинке написано "увлажняет до 24 часов"
- на втором слайде обещают восстановление сухой кожи
- на картинке есть "без липкости"
```

---

## 5. Что Непонятно

```text
unclear_or_missing:
```

---

## 6. Инструкция Для ИИ

Собери из этого чистый `Product Passport`.

Правила:

- отдели видимые данные от предположений;
- не критикуй карточку;
- не анализируй визуал;
- не пиши первое впечатление;
- не придумывай отсутствующие характеристики;
- если неясно, ставь `unclear`;
- текстовые обещания селлера сохрани как claims;
- цену, объём, рейтинг и отзывы подготовь для `price_value_snapshot`.
