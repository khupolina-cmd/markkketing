# Price Value Snapshot

Статус: снимок цен и доверия на момент анализа.

Не стратегия. Не вывод. Только видимые якоря для матрицы.

```text
snapshot_date:
marketplace:
search_query:
currency:
source_mode: screenshot / copied_page_text / manual / mixed
```

---


Цена за единицу считается только если понятна база сравнения:

```text
100 мл / 100 г / 1 кг / 1 штука / 1 таблетка / 1 применение
```

---

## 3. Что Не Заполняем

Не заполняем:

- точную динамику выдачи;
- региональную доставку;
- прогноз сроков;
- свежесть отзывов без дат.

Если это не лежит на поверхности, пропускаем.
