# Template: Matrix Conclusions

Этот файл создаётся после мастер-матрицы героя.

Путь:

```text
04_matrix_conclusions/hero/{hero_product_key}/04__{hero_product_key}__matrix_conclusions.md
```

Главное правило:

```text
здесь не строим ещё финальную стратегию
здесь фиксируем, что покупатель увидит / может считать из матрицы
и как это можно интерпретировать
```
