# Промпт 06: Контекстная severity

## Назначение

Для каждого профиля или диагностической линзы пересчитать severity каждой негативной темы.

Контекстная severity может только повышать базовую.

## Входные данные

```text
weighted_signal_map: [из промпта 01 — только негативные темы]
persona_profiles: [из промпта 05]
heuristic_bridges_map: [из промпта 06b]
```

## Инструкция

Для каждой темы спросить:

`насколько этот сигнал становится важнее для данной линзы?`

Учитывать:

- мотив;
- главный страх;
- сценарий использования;
- цену ошибки;
- эвристический мост.

### Типовые повышения

- подарок + мятая упаковка -> low -> high
- originality doubt -> low -> medium/high
- чувствительность к аллергии + любой сигнал реакции -> до critical
- высокая цена ошибки + слабый эффект -> medium -> high

## Формат выхода

```text
=== CONTEXTUAL_SEVERITY_MAP ===

Тема | Базовая | [Линза 1] | [Линза 2] | Bridge uplift | Обоснование повышений
```

## Правила

- `critical` не понижается
- понижать severity нельзя
- каждое повышение должно быть объяснено
- если повышения нет, остаётся базовая severity

## Анти-галлюцинаторные правила

1. Не повышать без обоснования
2. Не игнорировать подтверждённый bridge uplift
3. Не придумывать связь между темой и линзой, если её нет
