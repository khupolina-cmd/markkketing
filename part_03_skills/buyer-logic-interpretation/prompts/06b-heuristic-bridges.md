# Промпт 06b: Эвристические мосты

## Назначение

Найти случаи, где слабый или средний по природе сигнал запускает более сильный вывод в голове покупателя.

## Входные данные

```text
final_signal_registry: [из skill 1]
weighted_signal_map: [из промпта 01]
persona_profiles: [из промпта 05]
```

## Инструкция

1. Найти темы, склонные запускать вторичный вывод:
   - упаковка;
   - внешний вид;
   - признаки небрежности;
   - агрессивные обещания;
   - конфликты "заявлен премиум / выглядит дёшево";
   - неясность использования.
2. Построить мост только если есть основание:
   - прямая цитата;
   - повторяемый паттерн;
   - устойчивая рыночная эвристика.
3. Для каждого моста указать:
   - bridge type;
   - воспринимаемый вывод;
   - кому особенно релевантно;
   - силу моста;
   - что именно он вскрывает.

## Формат выхода

```text
=== HEURISTIC_BRIDGES_MAP ===

Источник сигнала: [тема]
Bridge type: [имя]
Промежуточный вывод покупателя: [...]
Кому особенно релевантно: [...]
Сила моста: [weak/moderate/strong]
Основание: [...]
Что вскрывает: [...]
```

## Анти-галлюцинаторные правила

1. Не строить мост без основания
2. Не путать мост с новым сигналом
3. Лучше 3-7 сильных мостов, чем 20 фантазий
