# Базовая шкала severity

## Принципы

- 4 уровня: `critical`, `high`, `medium`, `low`
- Уровня `noise` не существует
- Severity не зависит от частоты
- Severity проставляется только для негативных сигналов
- Базовая severity не зависит от типа покупателя
- Базовая severity не равна воспринимаемой силе сигнала

Контекстная сила сигнала может усиливаться в Слое B через:

- покупательскую линзу;
- эвристический мост;
- цену ошибки.

---

## Уровни

### critical

Вред здоровью, безопасность, правовой риск.

Примеры:

- аллергия
- ожог
- отёк
- подмена состава
- просрочка
- плесень

Правило:

`critical` всегда остаётся `critical`.

### high

Полный провал основной функции товара.

Примеры:

- "совсем не увлажняет" для увлажняющего крема
- "не защищает от солнца" для SPF
- "стойкость отсутствует" для товара, где стойкость заявлена как ключевая

### medium

Заметная проблема, товар частично работает.

Примеры:

- слабый эффект
- неудобный дозатор
- слишком быстро заканчивается
- конфликт текстуры с ожиданием

### low

Мелкое неудобство, товар в целом работает.

Примеры:

- мятая упаковка
- плохая полиграфия
- неудобная крышка
- выгоревшая этикетка

Важно:

`low` не означает "никому не важно".

Это означает только, что по своей природе сигнал слабый.

---

## Что базовая severity не покрывает

Базовая severity не отвечает на вопрос:

`во что этот сигнал превращается в голове покупателя`

Для этого есть эвристические мосты.

Примеры:

- `мятая упаковка`
  - base severity: `low`
  - possible bridge: `originality_doubt`
  - perceived impact: может стать `medium/high` для части линз

- `дешёвый визуальный вид`
  - base severity: `low`
  - possible bridge: `premium_claim_conflict`
  - perceived impact: подрывает премиализацию

- `слишком агрессивное обещание`
  - base severity: не сама по себе
  - possible bridge: `trust_drop`
  - perceived impact: ощущение вранья и завышенных ожиданий

---

## Позитивные сигналы

Для позитивных сигналов severity не проставляется.

Для них оценивается только `evidence_strength`:

- `strong`
- `moderate`
- `weak`
- `suspect`
