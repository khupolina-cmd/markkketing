# Правила противовесного анализа

## Назначение

Противовесный анализ отвечает на вопрос: **можно ли ставить эту ось в ядро подачи?**

Для каждой оси критерия (запах, увлажнение, текстура, упаковка, цена, стойкость...) сопоставляются позитивные и негативные сигналы. Результат — вердикт по шкале.

---

## Что такое «ось критерия»

Ось критерия — это тематический кластер, объединяющий похожие сигналы.

Примеры осей:
- **Запах** — все сигналы об аромате, запахе, стойкости запаха
- **Увлажнение** — все сигналы о увлажняющем эффекте, коже после
- **Текстура** — сигналы о консистенции, ощущении на коже
- **Упаковка** — сигналы об упаковке, дозаторе, внешнем виде
- **Состав** — сигналы об ингредиентах, аллергии, реакции
- **Цена/качество** — сигналы о соотношении цены и результата

Одна тема может принадлежать нескольким осям, но в противовесе она учитывается по **основной оси**.

---

## Алгоритм анализа

### Шаг 1: Определить оси
Сгруппировать темы из `weighted_signal_map` по осям критериев.

### Шаг 2: Собрать позитив и негатив
Для каждой оси:
- Перечислить все **позитивные** темы с их count
- Перечислить все **негативные** темы с их count и severity

### Шаг 3: Рассчитать частотный баланс
```
balance = позитивных_упоминаний / негативных_упоминаний
```
Если негативных упоминаний 0 — баланс = ∞ (нет опровержений).

### Шаг 4: Определить max severity негативов
Взять максимальный уровень severity среди всех негативных тем по данной оси.

### Шаг 5: Вынести вердикт

---

## Шкала вердиктов

### ✅ Сильная опора
**Условия:**
- Частотный баланс ≥ 4:1 (позитив к негативу)
- Max severity негативов ≤ medium
- Evidence positives = strong

**Значение:** Можно ставить в ядро подачи. Аргумент сильный, риск низкий.

---

### ⚠️ Условно рабочее
**Условия:**
- Частотный баланс 3:1–4:1
- Max severity ≤ medium

**Значение:** Можно упоминать, но не делать центральным. Нужна осторожная формулировка.

---

### 🟡 Спорное
**Условия:**
- Частотный баланс 2:1–3:1
- ИЛИ max severity = high при любом соотношении частот

**Значение:** Нельзя ставить в ядро без серьёзных оговорок. Высокий риск разочарования.

---

### 🔴 Нельзя в ядро
**Условия:**
- Частотный баланс < 2:1
- ИЛИ max severity = critical при любой частоте негатива

**Значение:** Нельзя делать центральным аргументом. Если упоминать — только с явными оговорками.

---

### ⚫ Нет данных
**Условия:**
- Менее 5 сигналов по оси (позитивных + негативных суммарно)

**Значение:** Данных недостаточно для вердикта. Не делать ставок ни в одну сторону.

---

## Критичное правило

> **Один `critical` негатив блокирует ось независимо от частоты позитива.**

Даже если у товара 200 позитивных отзывов об увлажнении и 1 отзыв об аллергической реакции — ось «состав/безопасность» получает вердикт 🔴.

Причина: один случай вреда здоровью несопоставим со статистическим позитивом. Правовой и репутационный риск неприемлем.

---

## Учёт evidence_strength

При оценке позитива учитывается `evidence_strength`:

| Evidence | Вес при подсчёте баланса |
|---|---|
| strong | 1.0 (полный вес) |
| moderate | 0.8 |
| weak | 0.5 |
| suspect | 0.2 |

Если большинство позитивных сигналов имеют evidence = suspect, реальный баланс хуже, чем кажется по количеству.

---

## Формат выхода: counterweight_report

Для каждой оси:

```
=== Ось: [название] ===

ПОЗИТИВ:
- [тема]: N упоминаний, evidence = [strength]
- [тема]: N упоминаний, evidence = [strength]
Итого позитивных упоминаний: N

НЕГАТИВ:
- [тема]: N упоминаний, severity = [уровень], evidence = [strength]
- [тема]: N упоминаний, severity = [уровень], evidence = [strength]
Итого негативных упоминаний: N

БАЛАНС: N:N (позитив:негатив)
MAX SEVERITY негативов: [уровень]
ВЕРДИКТ: [✅/⚠️/🟡/🔴/⚫]
РЕКОМЕНДАЦИЯ: [краткая рекомендация]
```

**Пример:**

```
=== Ось: Увлажнение ===

ПОЗИТИВ:
- «кожа стала мягкой»: 47 упоминаний, evidence = strong
- «хорошо увлажняет»: 31 упоминание, evidence = moderate
- «кожа не шелушится»: 12 упоминаний, evidence = moderate
Итого позитивных упоминаний: 90

НЕГАТИВ:
- «не увлажняет совсем»: 18 упоминаний, severity = high, evidence = strong
- «эффект пропадает быстро»: 9 упоминаний, severity = medium, evidence = moderate
Итого негативных упоминаний: 27

БАЛАНС: 3.3:1 (позитив:негатив)
MAX SEVERITY негативов: high
ВЕРДИКТ: 🟡 Спорное
РЕКОМЕНДАЦИЯ: Нельзя ставить «глубокое увлажнение» в центр без оговорки. Значительная доля не получает обещанного результата.
```
