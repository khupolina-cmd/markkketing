# Гайд Creative Reserve

## Что такое Creative Reserve

Creative Reserve — это блок skill 2, который превращает интерпретированные сигналы в **практический материал для дальнейшей работы**: копирайтинга, заголовков, фото-концептов и стратегии позиционирования.

Creative Reserve не формулирует финальные рекомендации по карточке (это skill 3). Он поставляет **сырьё**: идеи, формулировки, направления.

---

## 6 способов использования данных

---

### 1. Генератор заголовков

**Что брать:** Сигналы из категории «ожидания» (из skill 1).

**Откуда:** `final_signal_registry`, категория ожидания.

**Как использовать:**
- Ожидание покупателя → потенциальная формулировка для оффера
- Используется язык покупателя, не маркетинговый штамп
- Каждое ожидание = потенциальный заголовок в формате «обещание»

**Примеры:**

| Ожидание из данных | Идея заголовка |
|---|---|
| «хочу, чтобы кожа была мягкой весь день» | «Мягкость с утра и до вечера» |
| «ищу что-то для подарка маме» | «Подарок, который она откроет снова» |
| «боюсь, что не поможет при очень сухой коже» | «Работает даже для очень сухой кожи» |

---

### 2. Карта возражений

**Что брать:** Сигналы из категории «сомнения» (из skill 1) + негативные темы с высоким deterrence_score.

**Откуда:** `final_signal_registry`, категория сомнения + `buyer_deterrence_map`.

**Как использовать:**
- Каждое сомнение = возражение, которое карточка должна снять
- Высокий deterrence_score = приоритет в снятии возражений
- Формат: «Возражение → что нужно сказать в карточке»

**Примеры:**

| Сомнение из данных | Что нужно снять в карточке |
|---|---|
| «А вдруг не подойдёт для чувствительной кожи?» | Указать протестированность / дерматологическую проверку |
| «Боюсь, что быстро закончится» | Указать объём и примерный расход |
| «Не знаю, подойдёт ли в подарок» | Показать упаковку крупным планом |

---

### 3. Язык покупателя

**Что брать:** Сильные позитивные сигналы (evidence = strong) с яркими формулировками.

**Откуда:** `final_signal_registry`, позитивные сигналы.

**Как использовать:**
- Живые цитаты и формулировки → основа для копирайтинга
- Не перефразировать: брать как есть или минимально адаптировать
- Избегать маркетинговых штампов — использовать слова реальных покупателей

**Примеры:**

| Живая формулировка из данных | Использование в копирайтинге |
|---|---|
| «кожа как будто напилась» | Метафора для описания увлажнения |
| «запах такой, что муж спрашивает что это» | Социальное доказательство для аромата |
| «наконец-то нашла то, что не щиплет» | Для аудитории с чувствительной кожей |

---

### 4. Инверсия клише

**Что брать:** Обещания производителя с вердиктом 🟡 или 🔴 (спорные или проблемные).

**Откуда:** `promise_safety_map` + обещания конкурентов (если известны из данных).

**Как использовать:**
- Что все обещают, но это спорно → можно отличиться, отказавшись от этого клише
- Вместо стандартного обещания — неожиданное честное заявление
- «Мы не обещаем X, но зато Y» — работает сильнее, чем ещё одно X

**Примеры:**

| Всеобщее клише | Инверсия |
|---|---|
| «Невидимая текстура» (спорно в данных) | «Текстура ощущается — значит, работает» |
| «Стойкость 24 часа» (опровергается) | «Освежайте по необходимости — это честно» |
| «Для всех типов кожи» (generic) | «Создан для тех, кто устал от «подходит всем»» |

---

### 5. Визуальные подсказки

**Что брать:** Сигналы из категории «сценарии» (из skill 1).

**Откуда:** `final_signal_registry`, категория сценарии.

**Как использовать:**
- Сценарий использования → идея для фото/видео контента
- Конкретная ситуация из жизни → визуальный образ

**Примеры:**

| Сценарий из данных | Идея для фото |
|---|---|
| «беру с собой в командировку» | Флакон в поездке, на столе отеля |
| «использую после душа перед сном» | Вечерний ритуал, мягкий свет |
| «дочка попросила такой же» | Мама и дочь, совместное использование |
| «открываю как маленький праздник» | Распаковка, крупный план упаковки |

---

### 6. Донор-категории

**Что брать:** Сигналы сравнения с другими категориями или брендами (из данных).

**Откуда:** `final_signal_registry`, любые сравнения или упоминания аналогов.

**Как использовать:**
- Если покупатели сравнивают с другой категорией → заимствовать коды этой категории
- Коды донор-категории могут работать сильнее, чем стандартные коды своей категории

**Примеры:**

| Сравнение из данных | Донор-категория | Что заимствовать |
|---|---|---|
| «похоже на спа-процедуру» | SPA/wellness | Ритуальность, роскошь, «время для себя» |
| «как от косметолога» | Профессиональная косметология | Экспертность, клинический подход |
| «как у бабушки, натуральное» | Бытовая органика | Доверие, натуральность, традиция |

---

## Формат выхода: creative_reserve

```
=== CREATIVE RESERVE ===

1. ГЕНЕРАТОР ЗАГОЛОВКОВ
[Список 5–10 идей заголовков на основе ожиданий]

2. КАРТА ВОЗРАЖЕНИЙ
[Список сомнений → что снять в карточке, по приоритету deterrence_score]

3. ЯЗЫК ПОКУПАТЕЛЯ
[Список живых формулировок для копирайтинга]

4. ИНВЕРСИЯ КЛИШЕ
[Список: стандартное обещание → альтернативный нестандартный угол]

5. ВИЗУАЛЬНЫЕ ПОДСКАЗКИ
[Список идей для фото/видео контента из сценариев]

6. ДОНОР-КАТЕГОРИИ
[Список сравнений → категории → что можно заимствовать]
```
