# Формула deterrence_score

## Назначение

deterrence_score — числовая эвристика для **ранжирования** барьеров покупки. Показывает, что сильнее всего отталкивает покупателя.

**Важно:** Это не точная наука. Это инструмент ранжирования, не абсолютная метрика. Не округлять до красивых чисел.

---

## Формула

```
deterrence_score = severity_weight × frequency_factor × amplifier
```

---

## Веса компонентов

### severity_weight (базовая severity темы)

| Severity | Вес |
|---|---|
| critical | 10 |
| high | 6 |
| medium | 3 |
| low | 1 |

При расчёте для конкретного типа покупателя используется **контекстная severity** (не базовая).

---

### frequency_factor (доля упоминаний темы от общего массива)

| Доля | Фактор |
|---|---|
| >10% | 1.5 |
| 5–10% | 1.2 |
| 2–5% | 1.0 |
| <2% | 0.8 |

Доля считается от общего числа отзывов/записей в массиве (из stats skill 1).

**Важное исключение:** Для severity = critical frequency_factor применяется, но итоговый score не может быть ниже 8.0. Один critical-сигнал всегда попадает в топ карты отпугивания.

---

### amplifier (усилители)

Amplifiers зависят от того, нарушает ли сигнал ожидание покупателя и подтверждает ли его сомнение.

| Условие | Amplifier |
|---|---|
| expectation_violated (обещание было, и оно не выполнено) | ×1.5 |
| doubt_confirmed (покупатель сомневался, и сомнение подтвердилось) | ×1.3 |
| Оба условия одновременно | ×2.0 |
| Ни одного условия | ×1.0 |

**Как определить amplifiers:**
- `expectation_violated` = есть сигнал ожидания по этой теме в `final_signal_registry`
- `doubt_confirmed` = есть сигнал сомнения по этой теме в `final_signal_registry`

---

## Примеры расчёта

### Пример 1: «не увлажняет» для увлажняющего крема

- Тема: «не увлажняет»
- Severity: high → severity_weight = 6
- Доля в массиве: 15% → frequency_factor = 1.5
- Ожидание «хочу увлажнение» есть → expectation_violated
- Сомнения о недостаточном эффекте есть → doubt_confirmed
- Amplifier = ×2.0

```
deterrence_score = 6 × 1.5 × 2.0 = 18.0
```

---

### Пример 2: мятая упаковка (общая карта)

- Тема: «мятая упаковка»
- Базовая severity: low → severity_weight = 1
- Доля в массиве: 8% → frequency_factor = 1.2
- Ожиданий об упаковке в данных нет → expectation_violated = нет
- Сомнений об упаковке нет → doubt_confirmed = нет
- Amplifier = ×1.0

```
deterrence_score = 1 × 1.2 × 1.0 = 1.2
```

---

### Пример 3: мятая упаковка (для типа «Дарительница»)

- Тема: «мятая упаковка»
- Контекстная severity для дарительницы: high → severity_weight = 6
- Доля в массиве: 8% → frequency_factor = 1.2
- Ожидание «красивый подарок» есть → expectation_violated = да
- Amplifier = ×1.5

```
deterrence_score = 6 × 1.2 × 1.5 = 10.8
```

Для дарительницы мятая упаковка — серьёзный барьер (10.8 против 1.2 в общей карте).

---

### Пример 4: аллергическая реакция

- Тема: «аллергия»
- Severity: critical → severity_weight = 10
- Доля в массиве: 1% → frequency_factor = 0.8
- Ожидание безопасного состава есть → expectation_violated = да
- Сомнение в составе есть → doubt_confirmed = да
- Amplifier = ×2.0

```
deterrence_score = 10 × 0.8 × 2.0 = 16.0
```

Исключение применяется (score не ниже 8.0), итог = 16.0 — попадает в топ.

---

## Интерпретация score

Нет фиксированных порогов — это инструмент ранжирования. Значения сравниваются друг с другом:

| Диапазон (ориентир) | Интерпретация |
|---|---|
| >15 | Топ-барьер — сильнейший блокер |
| 8–15 | Серьёзный барьер — влияет на решение |
| 3–8 | Умеренный барьер — замечают, но не обязательно блокирует |
| <3 | Слабый барьер — мелкое неудобство |

## Формат выхода: deterrence_map

### Общая карта

```
=== КАРТА ОТПУГИВАНИЯ (общая) ===

# | Тема | N упом. | % | Severity | Amplifiers | Score | Вердикт
1 | [тема] | N | X% | [уровень] | [да/нет] | X.X | [топ-барьер / серьёзный / умеренный / слабый]
2 | ...
```

### Карта для типа покупателя

```
=== КАРТА ОТПУГИВАНИЯ для типа: [название] ===

# | Тема | Базов. severity | Контекст. severity | Score | Δ vs общая | Вердикт
1 | [тема] | [базовая] | [контекстная] | X.X | +/- X.X | [вердикт]
2 | ...
```

`Δ vs общая` показывает, насколько сильнее этот барьер для данного типа по сравнению с общей картой.
