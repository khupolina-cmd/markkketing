# Пример: prisms-library.md

> **Что это:** Файл для переноса кластеров на следующий товар.  
> Создаётся автоматически Skill 2 в конце каждого прогона (Секция 7 вывода).  
> Передай этот файл в следующий прогон Skill 2 для режима «с багажом».

---

## Шаблон

Скопируй блок ниже — он будет заполнен Skill 2 автоматически.

```
product_key: [ключ предыдущего товара]
category: [категория]
date: [дата прогона]
total_reviews: [N]

КЛАСТЕРЫ:

### [Название кластера 1]
уверенность: ●/◐
информативных_отзывов: N
драйверы:
  - 🟢 [сигнал]
  - 🟢 [сигнал]
терпимые_минусы:
  - 🔴 [сигнал]
стоп_факторы_гипотеза:
  - 🔴 [сигнал]
явные_мотивации: [список или «нет»]
якорные_отзывы: [ID через запятую]

[... следующие кластеры ...]

CONFLICT_MAP:
  [параметр]: [кластер А] vs [кластер Б]

КАЛИБРОВКА:
  [сигнал]: восторженно N, трезво M
```

## Пример заполненного файла

*(На основе парфюмированного крема для рук — категория «уход за руками»)*

```
product_key: repose-flower-hand-cream-v1
category: уход за руками (парфюмированные кремы)
date: 2025-04-01
total_reviews: 209

КЛАСТЕРЫ:

### Эстетика + Запах
уверенность: ●
информативных_отзывов: 38
драйверы:
  - 🟢 упаковка (внешний вид, тюбик, коробка)
  - 🟢 аромат (первое впечатление)
  - 🟢 ощущение «подарочности»
терпимые_минусы:
  - 🔴 среднее увлажнение (упоминают в 12 отзывах, но итоговая оценка позитивная)
стоп_факторы_гипотеза:
  - 🔴 повреждённая упаковка при доставке (не встречается в кластере)
явные_мотивации: «купила в подарок» — #12, #47, #98; «взяла себе как награду» — #63
якорные_отзывы: 12, 47, 63, 98, 115, 132

### Реальный уход
уверенность: ●
информативных_отзывов: 29
драйверы:
  - 🟢 увлажнение (долго держится)
  - 🟢 текстура (впитывается, не липнет)
  - 🟢 результат через неделю
терпимые_минусы:
  - 🔴 упаковка без коробки (есть в 8 отзывах, не остановила)
стоп_факторы_гипотеза:
  - 🔴 «закончился слишком быстро» (не встречается в кластере)
  - 🔴 «запах химический» (не встречается — 1 отзыв вне кластера)
явные_мотивации: нет
якорные_отзывы: 8, 22, 55, 89, 104, 167, 188

### Запах как ритуал
уверенность: ◐
информативных_отзывов: 14
драйверы:
  - 🟢 стойкость аромата
  - 🟢 характер запаха (цветочный, пудровый)
  - 🟢 ощущение после мытья рук
терпимые_минусы:
  - 🔴 небольшой объём (упоминают в 5 отзывах, но не остановил)
стоп_факторы_гипотеза:
  - 🔴 «запах исчезает через час» (не встречается в кластере)
явные_мотивации: нет
якорные_отзывы: 31, 76, 109, 145

CONFLICT_MAP:
  аромат: «Эстетика + Запах» (хочет яркий, заметный) vs «Реальный уход» (хочет нейтральный или лёгкий)
  объём: «Реальный уход» (хочет большой, расходный) vs «Эстетика + Запах» (малый объём = подарочный формат, ок)

КАЛИБРОВКА:
  🟢 аромат: восторженно 52, трезво 18 (#8, #22, #89). Реальность ближе к трезвой.
  🟢 увлажнение: восторженно 34, трезво 29 (#55, #104, #167). Результаты сильно варьируются.
  🔴 липнет после нанесения: упомянули 4 отзыва. Реальная доля недовольных может быть выше.
```

## Правила работы с файлом

- **Не редактировать вручную** — файл заполняется автоматически Skill 2
- **Хранить историю** — добавляй новые прогоны в конец файла под новым `product_key`
- **Кластер переносится** в следующий прогон только если подтверждён в 3+ товарах
- **Проценты не использовать** — только количество отзывов
