# Category Rules

Этот skill не переразмечает сигналы.

Он работает только с уже размеченной категорией.

Задача:

- отфильтровать нужную категорию;
- сгруппировать сигналы;
- посчитать частоты;
- поднять полные цитаты по ссылкам.

Для сравнений важно сохранять `X`.

Для ожиданий и сомнений — сохранять почти-цитатные формулировки.
