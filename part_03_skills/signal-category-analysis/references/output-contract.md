# Output Contract

## filtered_signals

Все сигналы нужной категории.

## themes

Список тем или почти-цитатных формулировок.

## counts

Для каждой темы:

- `R_signal`
- `% B_total`
- `% B_clean`

## quote_context_block

Для каждой темы:

- `product_key`
- `unit_index`
- `signal_fragment`
- `full_quote`

## category_report

Готовая сводка по одной категории для одного или нескольких товаров.
