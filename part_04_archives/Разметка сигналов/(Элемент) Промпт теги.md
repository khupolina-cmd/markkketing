{СИГНАЛЫ}

Ниже я даю тебе уже зафиксированные сигналы в формате:

страница/отзыв | категория | код сигнала

Пример:

2/7 | негатив | мят.уп.

Твоя задача — не искать сигналы заново, а работать только с уже размеченными сигналами, которые вставлены в блок {СИГНАЛЫ}.

#### Что нужно сделать

Для каждого сигнала разверни полный отзыв / вопрос целиком.

Внутри полного текста жирным выдели только тот кусок, который соответствует этому сигналу.

Если в одном отзыве несколько сигналов, разворачивай их по отдельности, а не одним блоком.

После полного разворота сгруппируй сигналы в темы по 2–4 слова.

Посчитай частоты тем и отсортируй по убыванию.

Посчитай B = общее число единиц в PDF (вопросы + отзывы).

Для каждого финального тега посчитай % = (N / B) × 100.

#### Правила

не переопределяй категорию сигнала

не ищи новые сигналы

не додумывай другие куски

работай только с теми сигналами, которые вставлены в блок {СИГНАЛЫ}

если один и тот же отзыв разворачивается по нескольким сигналам, каждый раз показывай тот же полный текст, но с новым выделением

не сокращай цитаты

не пересказывай отзыв своими словами

#### Формат ответа

#### A. Все сырые сигналы

Для каждого сигнала:

страница/отзыв | категория | код сигнала

— «полный отзыв / вопрос, где **жирным выделен нужный кусок**» — стр.

Если в одном отзыве 3 сигнала, покажи этот отзыв 3 раза, каждый раз с новым выделением.

#### B. Все темы

После полного списка сигналов сделай сводку:

тема — N

— «полная цитата с **выделенным сигналом**» — стр.

— «полная цитата с **выделенным сигналом**» — стр.

и так далее, сколько реально найдено.

#### C. Редкие, но критичные

Если среди сигналов есть критичные (вред, аллергия, опасно, брак, поломка, протечки, разбитое, не то, подмена), покажи их отдельно:

«полная цитата с **выделенным сигналом**» — стр. — дата (если есть) / [даты нет]

#### D. Ручной разбор

Если среди присланных сигналов есть блок ручной разбор, покажи его отдельно без перераспределения.

Если ручного разбора нет — ничего не пиши.

#### E. Динамика по времени

если в сигналах / цитатах есть даты → 1–3 строки “раньше vs сейчас”

если дат нет → NO DATA

#### Финал

После полного результата выведи последней строкой только теги:

TAGS: тег (N/B, %); тег (N/B, %); ...

Не добавляй служебные/мета-теги.
